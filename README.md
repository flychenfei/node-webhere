# Use
Start hapijs webserver on port 9000 and start browser when complete
```
cd ~/Dev/amazingwebapp
_wh 9000
```

Start hapijs webserver on port 8888(default) and start browser when complete
```
cd ~/Dev/superwebsite
_wh
```
